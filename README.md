# devops-product-service

## Overview
This repository contains the Product Service for the DevOps final project e-commerce platform.

## Responsibilities
- Provide REST API endpoints for product catalog operations
- Serve product data to the frontend and other services

## Tech Stack
- Node.js
- Express

## Branching Strategy
This repository follows the Git Flow model:
- `main`: production-ready code
- `develop`: integration branch for ongoing development
- `feature/*`: new features
- `release/*`: release preparation
- `hotfix/*`: urgent production fixes
